Game: Totally Rad (NES, 1990)20130318
I played the first level of this for a bit. I found it to be
pretty mediocre, but not insultingly bad. The first level is a
very boring straight-line with slopes that you strangely have to
jump over and can't walk up. There's no structure on the powers,
which you can access right away. I remember seeing the box of this
game and playing it when I was younger, but it didn't catch me and
I don't feel like I need to give it a lot of respect and
attention.
