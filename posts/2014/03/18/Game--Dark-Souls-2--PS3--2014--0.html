Game: Dark Souls 2 (PS3, 2014)20140318
I started this. I'll be playing "No leveling / Melee only / No
armor", but I will be allowed to upgrade anything else and use
shields. I created a Warrior, because it seems like that class has
the highest individual attack based stat, so I'll be able to make
the most of weapons. I played up through the second bonfire in the
Forest of Giants. So far, I'm not finding it super hard or much
different than the other games. I played it safe and have only
died twice, and one time was really dumb, trying to fight the huge
guy going to the bathroom near the start of the game.

These games are very fun and mechanically rewarding. I'm excited
about my challenge run. I tried to do it after the fact with Dark
Souls, but didn't feel like it was different enough to warrant
going through. I'm already building a map in my mind of things
that I want to investigate further and see how they
connect... exciting! The game does feel a bit more linear though,
because I only found one direction I could go in the beginning.
