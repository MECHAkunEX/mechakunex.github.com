Game: Sil (PC, 2012) (3)20170713
I made a lot of progress today. I went from 400ft to 650ft. It was
very tense and I had some crazy encounters. I'm learning a lot
about how things in the game work and having a very good time. I
kept track of all the things that I got so I'd remember:

400ft - Stealth 1 -> 3
400ft - Will 3 -> 9
400ft - Inner Light
400ft - Disguise

600ft - Perception 8
650ft - Listen
650ft - Will 6

I am abusing save states a lot, but I don't feel too bad about it,
because the game is so hard and kind of random in some ways. I
actually like it so much better, because it feels more like a big
puzzle that I'm figuring out how to get through.

My main strategy is to stealth around and if something is worth
fighting (high EXP reward for killing it), then I stealth kill it,
then run away and hide then do it again, or I launch arrows at it
from a distance before kiting again. It is very fun!
