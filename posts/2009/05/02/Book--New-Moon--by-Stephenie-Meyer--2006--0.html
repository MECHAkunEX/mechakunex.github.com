Book: New Moon, by Stephenie Meyer (2006)20090502
I was very impressed how different this was from
<cite>Twilight</cite>, both in style and plot. There was definitely
no surprise for the "big reveal", because I knew it was going to
happen before opening the book (based on my intuition, not any
spoiler.) BUT, all the details and everything that happened after
that was very interesting and surprising. The one problem with this
book, IMHO, is that it doesn't stand alone as well as
<cite>Twilight</cite>, so I really feel like I must read the rest
of the books, whereas after <cite>Twilight</cite> I could have gone
either way.
