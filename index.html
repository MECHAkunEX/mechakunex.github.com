Game: Lightning Returns: Final Fantasy XIII (PS3, 2014) (4)20140214
I played for about four hours. I felt like I mostly just ran
around the Wildlands as I tried to do the Animal Crossing
minigames to get the Chocobo back to health. I found the Dryads to
be incredibly difficult, because I didn't have any magic equipped
except for Elementa, which healed them. I think I'm getting to the
point in the game where I can't use it anymore because most
enemies are healed by some element. 

I was able to get the Etro village even though I didn't heal my
Chocobo all the way because I tried very hard on the jump. This
let me fight the boss that changes which kind of magic it
absorbs. It was a really hard boss and I died the first time. It
forced me to revise my schemata and I came up with a much more
powerful and diverse combination. My Sentinel role now is
basically impervious because of the hood you get from Snow,
although that means I can't attack all. I think I will compensate
that by adding some buffing or debuffing to it. Then I changed to
a different Ravager setup with a wider variety of spells. I was
able to beat the boss in no time, but when I got back to the
village to turn it in, the leader was gone. Ugh, so annoying.

I took a slight diversion to go see Vanille, which was cool, but I
was surprised that there wasn't more to do on that line. I get a
feeling from that part like Valkyrie Profile where the true way to
"win" the game is do ignore what you're supposed to do and find
some other secret path. I'm doubtful that this game is so
subversive though.

I'm on my way to the Temple of Etro now though. I can't decide if
I'll do Sazh next or go to the Dead Dunes for the first time.

Total time: 15:01

Game: Lightning Returns: Final Fantasy XIII (PS3, 2014)20140211
I played this for about five hours today after it came in the
mail. I was a huge fan of the previous games in the series, so I
was definitely looking forward to this.

The installation process took a long time and I wish I started it
earlier in the day, before I wanted to play. I like that you can
get the Japanese voice-over track, but I don't think I want to use
it, because I'm used to everyone's voices.

The game starts pretty nice and I'm enjoying the battle system. I
think it is a little weird that the added back in the HP meta-game
that was gone in previous FF13s. I find the random battles to be
incredibly easy, but in the graveyard on the main quest, they were
incredibly hard and I had to run from quite a few of them. I hope
that I can go back and train a bit on them.

The time constraint in the game is very oppressive to me, because
it feels like I don't have enough time to do what I want and
definitely not enough to grind. I spent about 8 hours of game time
looking for the final code that I would never have been able to
find because it could only be found in a place that was sealed
during the day. I wish I had used that time to go to one of the
other regions or something.

I'm not finding the customization to be super useful. I've got one
schemata for "Ravager", "Commando", and "Sentinel" each, and then
I'm just optimizing their pieces based on what new things I get. I
feel like I want a little bit more customization and a better
split between the powers I want and the way I want Lightning to
look. I'm sure this will get better as the game progresses, but
it's something I'm noticing now.

I really like the game though!

Total time: 4:57
