Game: DoomRL (PC, 2012)20170710
This is a fun game, but it has a similar problem to many
roguelikes: I don't feel like I am progressing when I play. It
seems like success is mostly random, so there's not that much
strategy or skill to develop, and there's nothing inside of the
game to acquire to make my character stronger, so it is only as
good as the second-to-second play experience. The user interface
is ever so slightly annoying and I feel like most of the strategy
is obvious, so there's not a lot to think about during each
move. This makes it more like a flashy version of something like
Sudoku. The game is definitely extremely well made and I
appreciate it.

Total time: 1:10
